# GOLLMax Mixture Regression – COVID-19 Application

This repository contains the code and data used in the article:

**Prataviera, F., Hashimoto, E. M., Ortega, E. M. M., Cordeiro, G. M., Cancho, V. G., & Vila, R. (2023).**  
*A new flexible regression model with application to recovery probability Covid-19 patients*.  
*Journal of Applied Statistics*.  
https://doi.org/10.1080/02664763.2022.2163229

## Overview

This repository provides the scripts and data used to reproduce the analyses presented in the article, including model fitting, simulations, and application to COVID-19 recovery data.

## Use of code and data

The code in this repository may be freely used, modified, and adapted for academic, educational, and research purposes.

The data made available here may also be used for reproducibility and research purposes. However, because the dataset used in the application was obtained from **Dong et al. (2020)**, any use of these data must also acknowledge the original data source.

## Citation requirement

If you use the code, data, or any material from this repository, please cite:

1. **The main article of this repository**
2. **The original data source: Dong et al. (2020)**
